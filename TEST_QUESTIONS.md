## 🐛 COMMON ISSUES

**Issue:** Bot says "I encountered an error"
**Fix:** Check that `corona_data.py` is in the same folder as `app.py`

**Issue:** Bot gives old information
**Fix:** Clear browser cache or use incognito mode

**Issue:** Bot is slow
**Fix:** First request on Render takes 30 seconds (cold start), subsequent requests are fast
